##预估网站访问量（我们第一个机器学习应用）

目的：预测一个网站的访问量

##数据格式
数据格式：  x  y  以Tab分割
 
- x 代表某一个小时  
- y 代表那一个小时内的Web访问数  


##清理数据
发现y值存在8个nan值，删除



##选择模型和学习算法

####近似误差

    def error(f, x, y):
        return sp.sum((f(x) - y)**2 )

近似误差是衡量一个模型是否正确的一个标准，这里是预测值与真实值的差的平方
